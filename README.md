# Leafpad-NixOS

Build and install Leafpad 0.8.17 on NixOS without using flakes.

## Installation

Clone the repo:

```bash
git clone https://github.com/OnlyLinux71/Leafpad-NixOS.git
cd Leafpad-NixOS

Build and Install:

nix-build
sudo nix-env -i ./result

Optional development shell:

nix-shell
