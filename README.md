Leafpad-NixOS

A simple Nix package and NixOS module for Leafpad 0.8.17.

git clone https://github.com/OnlyLinux71/Leafpad-NixOS.git
cd Leafpad-NixOS

# Build and install Leafpad
nix-build
sudo nix-env -i ./result


# Enter a Nix development shell for tinkering with Leafpad
nix-shell
